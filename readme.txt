== Description ==

This plugin used to add Puzzle Captcha to WordPress login and Register form and various WordPress Forms Plugins. Currently, it support only Contact Form 7 plugin.

Users must solve the puzzle captcha to confirm that they are not a robot before submitting login form of wp-admin, register form of wp-admin and contact form 7. Itâ€™s an easy way to solve the puzzle, just drag arrow from left to right to solve the puzzle and it is hard for bots to solve.

= How does it work? =

After activation of plugin, go to setting page of the plugin from where you can enable/disable the puzzle captcha for login form and register form of admin and there is an instruction of how to add puzzle captcha to Contact form 7 plugin.
